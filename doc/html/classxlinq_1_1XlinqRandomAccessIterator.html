Detailed Description

template<typename TElem>
class xlinq::XlinqRandomAccessIterator< TElem >

Iterator created from IEnumerator implementing C++ random access iterator concept.

This class implements C++ random access iterator concept and allows to use all read-only algrithms from <algorithm> header.
