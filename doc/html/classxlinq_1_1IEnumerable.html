Detailed Description

template<typename TElem>
class xlinq::IEnumerable< TElem >

Interface for enumerable.

This is basic enumerable interface which is used to obtain pointer to IEnumerator instance.

Member Typedef Documentation

Type returned by enumerator.

This member stores type which is returned during enumeration. It is designed to be not-constant and not-reference type.

Accesses enumerator.

This method allows to access pointer to enumerator, which may be used to traverse collection. Each of enumerator instances created from single collection should work separately (they should not have any impact on each other).
